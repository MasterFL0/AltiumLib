# AltiumLib
Проект библиотеки компонентов для САПР Altium Designer
## Инструкция по сборке и запуску
Запустить Altium Designer. 
В панели Components выбрать вкладку File-baised libraries preferences и добавить файл Library.IntLib в список библиотек.

Тестирование не предусмотрено
